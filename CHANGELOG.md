## 2026-04-05

### 初始需求
- 阅读测试题PDF，明确需求：开发一款类Vita Mahjong的麻将手游
- 核心要求：首页、游戏页、结果页，支持至少20关，手机可玩，24小时完成

### 方案选型
- **对比Unity vs Web方案**，最终选择Web方案（HTML5 + Phaser.js）
- 理由：开发效率高、即时预览、无需构建、Vercel一键部署
- 数据持久化：使用localStorage保存关卡进度

- **问题**：牌面类型随机分配，可能出现奇数张同类型牌，导致无法全部消除
- **修复**：所有牌面类型分配改为"先生成成对类型再打乱"
  - 第一层56张：按 `0,0,1,1,2,2...33,33` 循环生成后打乱
  - 第二层8-12张：同样按成对方式生成

  - 智能洗牌（`smartShuffle`）：同上，保证洗牌后每种类型仍成对
- **效果**：保证每张牌都有配对，一定能通过二销清除所有牌

### 入场动画系统
- **7种入场动画**，每次进入关卡随机播放一种：
  1. 左右交替飞入（按列奇偶从左/右飞入，按行从上到下延迟）
  2. 从上到下飞入（第0行先出现，逐行向下）
  3. 从下到上飞入（第7行先出现，逐行向上）
  4. 从右下角弹性放大
  5. 从左下角弹性放大
  6. 从右上角弹性放大
  7. 从左上角弹性放大
- **统一时长**：所有动画总时长约 1000ms
- **弹性效果**：手动两段式弹性（过冲→弹回），幅度减半
  - 飞入：过冲 15px + 80ms 弹回
  - 放大：过冲 1.07x + 80ms 弹回
- **锁定状态**：动画开始前即显示所有锁定牌的灰色蒙版
- **阴影渐显**：牌入场完成后，阴影 300ms 渐显
- **动画期间禁止点击**，结束后恢复交互并初始化自由状态

### 暗扣牌机制
- **功能**：每关随机将总牌数的 1/6 ~ 1/3 设为暗扣牌（背面 00.png 显示），保证偶数张以确保可配对消除
- **选中翻转**：点击暗扣牌执行水平翻转动画（shrink→换正面图→grow），阴影同步翻转，翻正后显示绿色选中效果
- **取消翻转**：再次点击执行反向翻转回背面，清除绿色 tint
- **消除动画**：暗扣牌在消除动画（起飞→靠近→缩小消失）期间保持正面状态，不再翻转
- **洗牌保持**：洗牌时暗扣牌保持背面显示，仅内部更新牌型数据
- **`resetTileState`** 中暗扣牌恢复背面
- **代码重构**：`createBoard()` 拆分为 `createBoard()` + `generateRandomLayout()`，为后续自定义关卡预留扩展点

### 提示功能增强：暗扣牌永久翻正 + 钟摆摇晃
- **提示翻正**：使用提示时，若匹配的牌是暗扣牌，执行水平翻转动画永久翻正为正面，作为普通牌参与游戏
- **黄色保持**：提示牌的黄色在各种场景下保持不消失（`recalculateAllFreeStatus`、消除回调、选中动画中均加 `hintTint` 检查）
- **钟摆摇晃**：提示牌每 1200ms 执行一次弹性摇晃（三次：小-4°→大±8°→小-4°，共 600ms），阴影同步跟随
- **选中不打断**：选中提示牌时变绿+放大，摇晃持续不中断；取消选中恢复黄色+继续摇晃
- **消除停止**：进入匹配消除动画时停止摇晃

- **层系统**：支持多层编辑，上层需要底层有支撑才能放置
- **保存**：关卡数据存储到 localStorage，游戏加载时自动读取定制布局

### 游戏逻辑适配 .5 位置
- **定制关卡加载**：`createBoard()` 检测 localStorage 中的关卡配置，有则调用 `generateLevelLayout()` 替代随机生成
- **`generateLevelLayout()`**：根据配置在指定位置创建牌，支持正格/边位/角位，白牌正面朝上，暗扣牌背面朝上
- **像素级碰撞检测**：`isTileFree()` 改为基于真实牌位置判断左右是否被挡住，不再依赖 boardGrid 整数索引
- **`hasSideNeighbor()`**：新增方法，按像素距离检测同层牌是否挡住左右侧
- **`refreshCoveredState()`**：新增方法，用真实包围盒重叠判断第一层牌是否被上层覆盖
- **深度层级提升**：选中牌深度 20000，碰撞动画牌深度 30000，确保自定义位置的牌也能正确置顶
- **消除解锁改为刷新**：消除上层牌后调用 `refreshCoveredState()` 统一刷新覆盖状态

### 首页新增定制关卡按钮
- 首页新增紫色「定制关卡」按钮，点击进入 EditorScene

### 深度层级重设计
- 深度公式改为 `depth = layer * 10000 + (row + col) * 100`
- row 和 col 权重相等，保证右压左、下压上同时成立
- 右邻牌 col +1 → depth +100，下邻牌 row +1 → depth +100
- 同层内越右、越下的牌层级越高，右下角牌压所有邻居
