# Mahjong Blast 玩法规则整理

## 游戏类型
Mahjong Blast 是一款**经典麻将接龙消除游戏**（Mahjong Solitaire / Tile-Matching），核心玩法是配对消除相同的麻将牌。

## 核心规则

### 1. 基本玩法
- 游戏目标：**清空棋盘上所有的麻将牌**
- 玩法：点击两张**相同的**麻将牌即可消除
- 牌必须处于"未锁定"状态才能被选中（见下方锁定规则）
- 消除所有牌即过关

### 2. 锁定规则（核心机制）
**这是与我们当前实现最大的区别！**

- 牌必须有**至少一侧是空的（开放边缘）**才能被选中
- 被其他牌**左右夹住**或**被上方牌覆盖**的牌是"锁定"的，无法选中
- 这是传统麻将接龙的核心规则——不是任意两张相同牌都能消除，必须是"自由牌"

#### 什么是自由牌（Free Tile）？
- 牌的**左侧或右侧**没有其他牌（即在边缘）
- 牌的**上方**没有其他牌覆盖（即在最上层）
- 同时满足以上条件的牌才是"自由牌"，可以被选中

### 3. 选中与消除流程
1. 点击一张自由牌 → 牌被选中高亮（蒙版效果）
2. 点击另一张**相同的**自由牌 → 两张牌消除
3. 如果两张牌不相同 → 选中状态取消，两张牌恢复原状
4. **每次只能选中两张牌**，不会出现选中多张的情况

### 4. 层叠设计
- 牌面布局不是平面的，而是有**层叠（stacking）**效果
- 某些牌会压在其他牌上面，被压住的牌无法选中
- 玩家需要**提前规划路径**，先消除上层/边缘的牌，逐步打开下层牌

### 5. 无时间限制
- 没有倒计时或强制操作
- 玩家可以按照自己的节奏思考和操作
- 适合休闲放松

## 道具系统

### 1. 提示（Hint）
- 功能：高亮显示一对当前可以配对消除的牌
- 次数有限（默认10次）
- 用完后图标变暗/禁用

### 2. 洗牌（Shuffle）
- 功能：重置当前剩余牌的位置（随机重新排列）
- 牌的种类和数量不变，只改变位置
- 用于打乱僵局，创造新的配对可能

### 3. 撤销（Undo）（部分版本）
- 功能：撤销上一步操作

## 与我们当前实现的对比

## 关键差异总结

**最重要的一点：我们的游戏缺少"锁定/自由牌"机制。**

当前实现中，只要两张牌相同就能消除，不管它们是否被其他牌遮挡。但 Mahjong Blast 的核心玩法是：
1. 牌有层叠关系（不是平面网格）
2. 只有"自由牌"（至少一侧空+上方无覆盖）才能被选中
3. 这个机制让游戏有了策略性——需要先消除上层牌来"解锁"下层牌
